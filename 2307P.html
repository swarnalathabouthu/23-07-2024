The Tags We Learnt Today: 23-07-2024

1.Bold Tag:

Make some text bold (without marking it as important).

2.Strong Tag

The content inside is typically displayed in bold.

3.Italic Tag

The content inside is typically displayed in italic.

4.Emphasized Tag:

The tag is used to define emphasized text. The content inside is typically displayed in italic.

5.Small Text Tag

The tag defines smaller text (like copyright and other side-comments).

6.SubScript Tag

Tag defines subscript text.

Example: WWW1

7.SuperScript Tag

tag defines superscript text.example:like WWW[1].

Example: X2+Y2

8.Ins Tag

Tag defines a text that has been inserted into a document. Browsers will usually underline inserted text.

9.Delete Tag

tag defines text that has been deleted from a document

10.Mark Tag

The tag defines text that should be marked or highlighted.
